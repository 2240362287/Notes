## 数据类型



```
基本：数字(number)、字符串(string)、布尔值(boolean)、空(null)、未定义(undefined) 

引用：Object、Array、Function 
```



- 数字  number 

- 字符串 string 
- 布尔值 boolean 
- 空 null 
- 未定义 undefined 
- symbol（ES6新增）
- 对象 object    (  数组 Array、函数 function、日期 Date   )



ES6引入了一种新的原始数据类型Symbol，表示独一无二的值

Symbol值不能与其他类型的值进行运算
